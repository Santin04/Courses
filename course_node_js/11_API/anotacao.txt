REST e RESTfull são padrões que devem ser seguidos para criar APIs de alta
qualidade

SPA são aplicações WEB que funciona com o metodo de alterar página alterando os
elementos DOM em vez de carregar outra página, esse tipo de aplicação está
crescendo cada vez mais, frameworks como React, Vue e Angular desenvolvem
aplicações SPA. Nesses caso o front end é separado do back end e a conexão
deles é realizada através de uma API

Vamos ter uma quantia maior de methods, sendo eles: Get, Post, Put, Delete e
Patch

Get: Receber registros
Post: Enviar registro
Put: Atualização completa de um registro
Delete: Deletar um registro
Patch: Atualização parcial de um registro

Códigos de status de resposta HTTP:
100 - 199 = Informação
200 - 299 = Sucesso
300 - 399 = Redirecionamento
400 - 499 = Erro do cliente
500 - 599 = Erro do servidor
